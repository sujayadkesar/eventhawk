# EventHawk Resources

## Logo

**File:** `eventhawk_logo.png`

**Size:** 1000x1000 px (recommended)
**Format:** PNG with transparent background
**Used in:**
- Main window header (left panel, top)
- Window icon (title bar, taskbar)
- About dialog

### To add the logo:

1. Save the EventHawk logo image as `eventhawk_logo.png` in this directory
2. Ensure it's a PNG file with transparent background
3. Recommended size: 1000x1000 px or larger (will be scaled automatically)

The application will automatically:
- Scale it to 80px height for the left panel header
- Scale it to 120px height for the about dialog
- Use it as the window icon (system tray)

No code changes needed after placing the file.
